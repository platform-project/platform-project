Troubleshooting

If your Platform install is not working correctly on your development machine, try the following command:

$ cd /platform && ./platform --reload
reloading done.

This command ensures that your platform install has all the correct filesystem permissions.

To confirm that your Platform CLI is working correctly, run the command:

$ platform --version
0.03 build 2014-03-28_10h33m57s
